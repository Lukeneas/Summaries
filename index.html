Prática

Navigation for Tabs
1. Add class in Elements to conect (js-tabname)
2. Make a function to add or remove class 'ativo' for each item accordingly with 'index' of other Element
3. Active this function in each item of other Element
4. Just active all this if the two Elements.length exist
5. Add ClassName to html in the 'index.html' with 'script' in top, to add if 'javascript' have active
Accordion List
1. Add class in Father
2. Make a click event in each first item
3. In callback, add class in each below item
Smooth Scroll
1. Make a click event in each link
2. Conect the link with section for href and id
3. section.scrollIntoView({behavior : 'smooth', block : 'start'}) or window.scrollTo({top: 'section.offsetTop', behavior: ''})
Obs :
CSS in HTML : scroll-behavior : smooth ( more fast )
Anime Scroll
1. Get all sections of site
2. Add event in window of scroll
3. Get offsetTop of each section
4. Active if offsetTop - 60%.window.Height < 0
5. Active 1x before, for to have something open in site
